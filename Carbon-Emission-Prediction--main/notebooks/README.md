# Notebooks Directory

## Execution Order:
1. **01_eda.ipynb** - Comprehensive exploratory data analysis
2. **02_preprocessing.ipynb** - Data cleaning and feature engineering
3. **03_arima_model.ipynb** - ARIMA model training
4. **04_prophet_model.ipynb** - Prophet model training
5. **05_lstm_model.ipynb** - LSTM model training
6. **06_model_comparison.ipynb** - Compare all models and export for dashboard

## Output:
- Trained models saved to `models/`
- Dashboard data saved to `data/dashboard/`
- Visualizations saved to `reports/figures/`
